## 説明文

Rust Drill は、Rust の基礎をクイズ形式で学ぶための学習アプリです。

所有権、借用、ライフタイム、型、エラー処理など、Rust 初学者がつまずきやすいテーマを短い問題で確認できます。問題を解いたあとに解説を読み、間違えた問題は復習タブで見直せます。

単語集では、Rust の学習中に出てくる用語を確認できます。隙間時間に少しずつ進めながら、理解した内容を定着させることを目的にしています。

本アプリはログイン、課金、広告 SDK、解析 SDK を使用していません。学習履歴、復習対象、表示設定、進捗は端末内に保存されます。
