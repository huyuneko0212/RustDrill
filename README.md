# Rust Drill (iOS / SwiftUI)

Rustの学習ハードルを下げるための、クイズ形式学習アプリ（iOS）です。  
隙間時間で **インプット + アウトプット** を高速に回すことを目的にしています。

## 目的

- Rust初学者がつまずきやすい概念（所有権・借用など）を段階的に学べる
- クイズ形式で理解度を確認できる
- 間違えた問題を復習して定着できる
- 将来的なオンラインDB移行を見据えて、Repositoryパターンで実装する

---

## 現在の実装状況（MVP）

###  データ基盤
- SwiftData を使ったローカル保存
- `questions.json` からの初期データ投入（Seed）
- Repository パターン導入（`QuizRepository` / `LocalQuizRepository`）

###  カリキュラム導線
- 大カテゴリ → 中カテゴリ → 小カテゴリ の階層表示
- 最下層カテゴリでは自動でクイズ画面へ遷移
- 各カテゴリに進捗バー表示（子カテゴリ配下を含む総問題数ベース）

###  クイズ機能（4択）
- 問題文 / コードスニペット表示
- 4択回答
- 正誤判定
- 解説画面表示
- 複数問題の「次へ」遷移

###  復習機能（needsReview）
- 不正解時に `needsReview = true`
- 復習タブで復習対象問題を一覧表示
- 復習クイズ実行
- 正解した問題は復習対象から外れる (`needsReview = false`)

###  進捗表示
- カテゴリごとに進捗を表示
  - 例: `3 / 10問`, `30%`
- 進捗率は「1回でも正解した問題」をクリアとして集計
